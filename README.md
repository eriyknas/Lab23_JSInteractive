# Лабораторная работа №23. Интерактивные веб-приложения на JavaScript
## Описание игры
Игра "Крестики-нолики" - классическая логическая игра для двух игроков на поле 3×3.
### Правила игры
- Игроки по очереди ставят на свободные клетки поля 3×3 знаки (X и O)
- Первый ход делает игрок за "X"
- Побеждает тот, кто первым выстроит в ряд 3 своих знака по горизонтали, вертикали или диагонали
- Если все 9 клеток заполнены, но победитель не определён - объявляется ничья
### Логика победы/поражения
- Победа: три одинаковых символа в одной линии 
- Поражение: противник собрал выигрышную комбинацию
- Ничья: все клетки заполнены, но выигрышных комбинаций нет
### Особенности реализации
- Игровое поле создаётся динамически через JavaScript
- Интерфейс адаптивный, с визуальными эффектами при наведении
- Есть кнопка перезапуска игры в любой момент
## Игровая механика
### Как определяется победитель?
Функция `checkResult()` перебирает все 8 выигрышных комбинаций из массива `winConditions`. Для каждой комбинации проверяется, что все три клетки заполнены одинаковым символом.
### Как обрабатываются клики?
При клике на клетку:
1. Проверяется, пуста ли клетка и активна ли игра
2. В клетку записывается символ текущего игрока
3. Обновляется массив `board`
4. Вызывается проверка результата
5. Если игра не окончена - происходит смена игрока
### Как работает перезапуск?
Функция `restartGame()`:
1. Сбрасывает `currentPlayer` на 'X'
2. Очищает массив `board` (все элементы становятся пустыми строками)
3. Устанавливает `gameActive = true`
4. Обновляет текст статуса
5. Очищает все клетки на экране
## Структура данных
### Массив board
```javascript
let board = ['', '', '', '', '', '', '', '', ''];
```
### Условия победы
```javascript
const winConditions = [
    [0, 1, 2], 
    [3, 4, 5], 
    [6, 7, 8],
    [0, 3, 6], 
    [1, 4, 7], 
    [2, 5, 8],
    [0, 4, 8], 
    [2, 4, 6]
];
```
### Состояние игры
```javascript
let currentPlayer = 'X';
let gameActive = true;
```
## Алгоритмы
### Алгоритм проверки победы
1. Перебираем все выигрышные комбинации из `winConditions`
2. Для каждой комбинации получаем значения из массива `board` по индексам
3. Если все три значения одинаковы и не пустые - победа
4. Устанавливаем `gameActive = false` и выводим сообщение
### Алгоритм определения ничьи
1. Используем метод `every()` для массива `board`
2. Проверяем, что все элементы не пустые
3. Если условие истинно - ничья
### Алгоритм смены игрока
```javascript
currentPlayer = currentPlayer === 'X' ? 'O' : 'X';
```
